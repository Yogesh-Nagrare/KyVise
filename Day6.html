HTML Media Tags Notes

1. Media Tag

In HTML5, media mainly refers to audio and video. They are used to embed sound, music, and videos into web pages.

Flow Diagram: How Media Works

2. Audio Tag (<audio>)

The <audio> tag is used to embed audio files.

  • controls → shows play, pause, volume controls.
  • autoplay → starts audio automatically.
  • loop → plays repeatedly.

Example:

3. Video Tag (<video>)

The <video> tag is used to embed videos.

  • controls → shows play, pause, volume controls.
  • autoplay → starts video automatically.
  • loop → repeats the video.
  • muted → starts video with no sound.
  • poster → sets a thumbnail before playing.

Example:

4. Control Attribute

When controls is present in <audio> or <video>, the browser automatically provides play/pause, volume, seek bar, and fullscreen options. Without controls, the user cannot interact with the media.
